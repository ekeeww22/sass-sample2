※ 네이밍 규칙

  • html 파일명 : 화면 ID 와 동일 하게. 단 점(.) 은 언더바(_) 로 대체

※ 주석 규칙

  • html : 태그 시작 지점 <-- 내용 --> , 태그 종료 지점 <-- // 내용 --> ,

※ CSS 규칙

  • css class name : 단어간 구분은 대시(-)로 구분
  • css code style : component, template 을 제외한 화면 ID 를 시작으로함 예) .A_CL_MA_0010 .section-01>ul>li
    => 화면ID는 모든 페이지의 <div class="content A_CL_MA_0010"> 에 class로 들어감

※ JAVASCRIPT 규칙

  • javascript version : ES5
  • function name : 카멜 표기법, 소문자의 동사 예) getValue()
  • 변수 name : 소문자의 명사 예) var value;
  • 상수 name : 대문자의 명사 예) var CONSTANT_VALUE;
